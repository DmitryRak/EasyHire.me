# EasyHire.me
Test scripts for easyhire.me

What need to be insalled:

1. JDK 1.8
2. Apache Maven 2.2.1 
3. Git 1.9.1

How to run:

1. git clone https://github.com/DmitryRak/EasyHire.me.git
2. cd EasyHire.me
3. mvn test

        Sample output:

Results :
